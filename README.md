# Librarium

Library Management System with:
- ASP.NET Core (Backend)
- Android App (WebView)

## Project Structure
- Backend/ → ASP.NET Core project
- AndroidApp/ → Android Studio project

## How to Run

### Backend
1. Open Backend in Visual Studio
2. Run the project
3. Note the URL (e.g. http://localhost:5059)

### Android App
1. Open AndroidApp in Android Studio
2. Run on device/emulator
3. Make sure backend is running

## Features
- Student login
- Book management
- Admin panel
- Email notifications
